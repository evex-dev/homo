# homo.js
数字を 114514 を使用して表すソースです。
これは元のソースのESM/TypeScript版です。高速化処理などを施しています。

## Playground
[https://lab.magiconch.com/homo/](https://lab.magiconch.com/homo/)

## 使用方法
```ts
import { homo } from "./homo.ts";

console.log(homo(1919))
```
